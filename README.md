Sistema de Gestión Bancaria - Arquitectura de Microservicios

Información General
Nombre del proyecto: BankCore - Sistema de Gestión Bancaria Distribuido
Tipo: Backend con arquitectura de microservicios para operaciones bancarias

Descripción del Sistema
Sistema bancario compuesto por dos microservicios que se comunican entre
sí para gestionar clientes y sus cuentas bancarias, permitiendo operaciones
como transferencias, consultas de saldo y movimientos.
